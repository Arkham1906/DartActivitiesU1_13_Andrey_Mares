# DartActivitiesU1_13_Andrey_Mares

Repositorio con las actividades de la Unidad 1 en Dart, donde se ven conceptos basicos del lenguaje y ejemplos practicos de consola.

---

## Contenido del repositorio

01_HolaMundo.dart  
Muestra un mensage de saludo en la terminal.

02_DataTypes.dart  
Explora los diferentess tipos de datos disponibles: numeros enteros, decimales, texto, valores boleanos, colecciones y diccionarios.

03_FinalConst.dart  
Ilustra las diferensias entre variables inmutables declaradas con final y constantes de tiempo de compilacion con const.

04_Coments.dart  
Muestra los diferentes estilos de anotaciones en codigo: de una linea, de multiples lineas y documentacion.

05_MathOperators.dart  
Implementa operasiones aritmeticas basicas y avanzadas.

06_Operators.dart  
Demuestra el uso de operadores de asignacion compuestos y el operador de coalescencia nula.

07_Read.dart  
Captura informacion ingresada por el usuario a travez de la entrada estandar.

08_HelloName.dart  
Solicita el nombre del usuario y genera un saludo persolanizado.

09_Sum.dart  
Realisa la adicion de dos valores numericos proporcionados por el usuario.

10_Concat.dart  
Demuestra tecnicas para unir cadenas de texto y embeber variables dentro de strings.

11_IfBasic.dart  
Implementa toma de deciciones mediante estructuras condicionales basicas.

12_Nested.dart  
Utilisa condicionales anidados para evaluar multiples condiciones relacionadas.

13_Menus.dart  
Presenta un sistema de opsiones interactivo para realizar calculos geometricos de areas y volumenes.

14_TablaMultiplicar.dart  
Crea y despliega tablas de multiplicasion utilizando estructuras de repeticion.

15_Calificacion.dart  
Traduse calificaciones entre el sistema de evaluacion estadounidese y mexicano.

16_Sumas.dart  
Acumula progresibamente valores numericos hasta recibir una se√±al de terminacion.

17_Labels.dart  
Ilustra el control de flujo en ciclos mediante instruciones de salto y continuacion.

18a_Funciones.dart  
Introduse la creacion y llamado de funciones reutilisables.

18b_Funciones.dart  
Construye tablas de multiplicasion mediante funciones parametrizadas.

19_Primos.dart  
Determina si un numero dado cumple con las caracteristicas de un numero primo.
